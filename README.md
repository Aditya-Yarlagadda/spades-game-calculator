# Spades Game Calculator
This project creates a spades game point calculator that can keep track of every player's score and calculate points every round based on the bet and their actual score.

## What it does
- Tracks scores for all players across multiple rounds
- Takes each player's bet and actual score
- Calculates their points based on the difference between their bet and score
- Keep a total on the side of the screen throughout the game
- Every action can be undone using the X Button.

## Built With
- Python
- Gradio

## How to run it
1. Install the requirements:
**pip install-r requirements.txt**
2. Run the app **SpadesGameScorer.py**
3. Open the local URL Gradio prints in your terminal
